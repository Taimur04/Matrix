MATRIX

সারি ম্যাট্রিক্স (Row matrix):
যে ম্যাট্রিক্সের কেবলমাত্র একটি সারি থাকে তাকে সারি ম্যাট্রিক্স বলে।
যেমন:[ 2 3 4 5 ]
কলাম ম্যট্রিক্স (Column matrix):
যে ম্যাট্রিক্সের কেবলমাত্র একটি কলাম থাকে তাকে কলাম ম্যাট্রিক্স বলে।
বর্গ ম্যাট্রিক্স ( Square matrix ):
যে ম্যট্রিক্সের সারি সংখ্যা ও কলাম সংখ্যা সমান তাকে বর্গ ম্যট্রিক্স বলে।
ম্যাট্রিক্সের প্রধান কর্ণ ( Principle Diagonal ):
কোনো বর্গ ম্যট্রিক্সের ১ম সারি ও ১ম কলামের সাধারণ ( common ) ভুক্তি বরাবর কর্ণকে প্রধান কর্ণ বলে। অর্থাৎ একটি বর্গ ম্যট্রিক্সের Aij, যেখানে i=j ভুক্তিগুলি বরাবর কর্ণকে প্রধান কর্ণ বলে।
প্রধান কর্ণ বরাবর ভুক্তিগুলির সমষ্টিকে ম্যাট্রিক্সটির ট্রেস বলে।
কর্ণ ম্যাট্রিক্স( Diagonal matrix ):
যে বর্গ ম্যাট্রিক্সের প্রধান কণের ( অর্থাৎ aij,i=j ) ভুক্তিগুলি ব্যতীত অন্যসব ভুক্তি শূন্য তাকে কর্ণ ম্যাট্রিক্স বলে।
স্কেলার ম্যাট্রিক্স ( Scalar matrix ):
যে কর্ণ ম্যাট্রিক্সের অশূন্য ভুক্তিগুলি সমান তাকে স্কেলার ম্যাট্রিক্স বলে।
দ্রষ্টব্য: সকল স্কেলার ম্যাট্রিক্সই কর্ণ ম্যাট্রিক্স।
অভেদক ম্যাট্রিক্স বা একক ম্যাট্রিক্স ( Identity matrix ):
কর্ণ ম্যাট্রিক্সের কর্ণস্থিত সকল ভুক্তি 1 হলে তাকে অভেদক ম্যাট্রিক্স বলে।
শূন্য ম্যাট্রিক্স ( Zero matrix ):
যে ম্যাট্রিক্সের সকল ভুক্তি শূন্য তাকে শূন্য ম্যাট্রিক্স বলে।
ট্রান্সপোজ বা বিম্ব বা রুপান্তরিত ম্যাট্রিক্স ( Transpose Matrix ):
একটি ম্যাট্রিক্সের সারিগুলি কলাম আকারে লিখে যে ম্যাট্রিক্সের পাওয়া যায় তাকে ম্যাট্রিক্সটির ট্রান্সপোজ ম্যাট্রিক্স বলে। অনুরূপভাবে সবগুলি কলামকে সারি আকারে লিখলেও ট্রান্সপোজ ম্যাট্রিক্স পাওয়া যায়।
A ম্যাট্রিক্সের ট্রান্সপোজ ম্যট্রিক্স A^T বা A^t বা A' দ্বারা সূচিত করা হয়।
প্রতিসম ম্যাট্রিক্স ( Symmetric Matrix ):
একটি বর্গ ম্যাট্রিক্সের ট্রান্সপোজ ম্যট্রিক্স একই হলে ( অর্থাৎ A^T=Aহলে ) তাকে প্রতিসম ম্যাট্রিক্স বলে।
বিপ্রতিসম ম্যাট্রিক্স (Skew Symmetric Matrix ):
একটি বর্গ ম্যাট্রিক্সের ট্রান্সপোজ ম্যাট্রিক্স তার ঋণাত্মক চিহ্নবোধক ( অর্থাৎ A^T=-A ) হলে তাকে বিপ্রতিসম ম্যাট্রিক্স বলে।
সমঘাতি ম্যাট্রিক্স ( Idempotent Matrix ):
একটি বর্গ ম্যাট্রিক্স A এর জন্য A.A =A হলে তাকে সমঘাতি ম্যাট্রিক্স বলে।
অভেদঘাতি ম্যাট্রিক্স ( Involuotory Matrix ):
একটি বর্গ ম্যাট্রিক্স A এর জন্য A.A=1 হলে তাকে অভেদঘাতি ম্যাট্রিক্স বলে।
শূন্যঘাতি ম্যাট্রিক্স ( Nilpotent Matrix ):
একটি বর্গ ম্যাট্রিক্স A এর জন্য A^n=0 হলে ( যেখানে n ক্ষুদ্রতম স্বাভাবিক সংথ্যা) ম্যাট্রিক্সকে শূন্যঘাতি ম্যাট্রিক্স বলে এবং n কে শূন্যঘাতির সূচক বলা হয়।
